Question (from finals page — Q10):
What is the difference between a simple LLM chatbot and an AI agent with tools? Give one concrete example of a "tool" and explain why it makes the agent more capable. Think about: what can an agent do that a plain LLM cannot? (e.g., look things up in real-time, run code, query a database, send an email...)

Expected: short comparison and one concrete tool example with rationale.

Answer:

A simple LLM chatbot generates text based only on its model weights and the prompt; it cannot take external actions or access live data. An AI agent with tools can call external systems (search, databases, code runners, APIs) and use their results to augment reasoning and perform real-world tasks.

Example tool — Web search or browser: by using a search tool, an agent can retrieve up-to-date information and verify facts in real time, then use that information to answer or take actions. This makes the agent more capable because it can access live data, follow links, and reduce hallucination versus relying solely on the model's static knowledge.

One-sentence summary:
Agents extend LLMs with action-capable tools, enabling live data access, stateful interactions, and real-world operations that plain chatbots cannot perform.
